# codewars-kata

Repositório com meus exercícios resolvidos no **Codewars**, focados em prática de lógica, fundamentos de programação e aprofundamento em Java.

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)
![Status](https://img.shields.io/badge/status-active-brightgreen?style=for-the-badge)

## Objetivo

Este repositório serve como registro da minha evolução em programação, com foco em:

- Lógica de programação
- Boas práticas em Java
- Estruturas de dados e algoritmos
- Resolução de problemas
- Escrita de código limpo e legível

## Estrutura do repositório

Os exercícios estão organizados por nível de dificuldade (kyu) e/ou tema:
```
codewars-kata/
│
├── 8-kyu/
├── 7-kyu/
├── 6-kyu/
└── README.md
```
